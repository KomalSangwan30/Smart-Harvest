About Smart Harvest

A simple project that uses machine learning to help farmers decide which crop to grow based on their soil and weather conditions.

📌 The Project

What is Smart Harvest?

Smart Harvest is a crop recommendation system. A farmer enters 7 details about their farm — things like how much nitrogen is in the soil, the local temperature, and how much rain falls every year.

The system then runs those numbers through 3 different machine learning models and picks the best one to give a crop recommendation. It also shows how confident the model is about its answer.

The goal was to build a full-stack web application that combines data science, a Django backend and a clean frontend — all working together.

⚙️ The Process

How Does It Work?

From entering your details to getting a recommendation — here is what happens behind the scenes

📝
STEP 1

You Enter Your Details

You fill in 7 values about your farm — nitrogen, phosphorus, potassium, temperature, humidity, soil pH and rainfall.

⚖️
STEP 2

Data is Normalised

The 7 numbers are scaled to the same range so the model can compare them fairly. This is called feature scaling and it makes the model more accurate.

🤖
STEP 3

ML Model Predicts

The best trained model looks at your values and predicts which of the 22 crops would grow best in those conditions. It was trained on 2,200 examples.

🌾
STEP 4

Result Shown

The crop name, confidence percentage, season, water requirement and a growing tip are all shown back to you instantly — no page reload needed.

🤖 Machine Learning

The 3 Models We Used

We trained 3 different models and picked the one with the best accuracy for predictions

📈

Logistic Regression

This is the simplest model. It draws a straight line to separate different crop groups from each other. Think of it like sorting fruits into baskets by drawing lines between them.

✓ Very fast to train
✓ Easy to understand
✗ Not great when data is complex
Accuracy: ~90%
🔍

K-Nearest Neighbors

This model looks at the 5 most similar examples from the training data and picks whatever crop appears the most. Like asking your 5 neighbours what they grow and going with the majority.

✓ Works well with clear patterns
✓ Easy to explain to anyone
✗ Gets slow on very large data
Accuracy: ~90%
🌲

Random Forest ⭐ Best

This model builds 100 decision trees and takes a vote from all of them. It is like asking 100 experts instead of just one — the majority decision is usually right. This gave us the best accuracy.

✓ Very accurate on complex data
✓ Rarely overfits
✗ Takes more memory than others
Accuracy: ~95% ✅

📊 Data Science

What Data Science Steps Did We Do?

Here are the main steps we followed to build and train the machine learning model

📦

1. Created a Dataset

We generated a dataset with 2,200 rows — 100 samples for each of the 22 crops. Each row has 7 features like nitrogen level, temperature, rainfall etc. This dataset was used to train our models.

🔢

2. Label Encoding

Machine learning models only understand numbers, not words. So we converted crop names like "rice" and "maize" into numbers like 0, 1, 2 etc. This is called Label Encoding.

⚖️

3. Feature Scaling

Nitrogen can be between 0 and 200 while rainfall can be 0 to 500. These different ranges confuse models. We used Standard Scaler to bring all features to the same scale so they are compared fairly.

✂️

4. Train and Test Split

We split the data — 80% for training the model and 20% for testing it. The model never sees test data during training, so the test accuracy tells us how it performs on completely new data.

🏋️

5. Model Training

All 3 models were trained on the training data. The model learns patterns — for example it learns that rice grows when nitrogen is high and humidity is above 80%. It saves these patterns for later use.

🏆

6. Picking the Best Model

After testing all 3 models we compare their accuracy scores. The model with the highest score is saved and used for live predictions. In our case Random Forest won with about 95% accuracy.

📋 Input Details

What Do the 7 Inputs Mean?

Here is a simple explanation of each value the system asks you to enter

Input What it Means Unit Range
🟤 Nitrogen (N) Helps the plant grow leaves. Get this from your soil test report. mg/kg 0 – 200
🟠 Phosphorus (P) Helps roots and flowers grow strong. Also from your soil test. mg/kg 0 – 200
🟡 Potassium (K) Makes the plant stronger and improves fruit quality. mg/kg 0 – 300
🌡️ Temperature The average temperature in your area throughout the growing season. °C -10 – 55
💧 Humidity How much moisture is in the air. Higher means more humid weather. % 0 – 100
🔬 Soil pH How acidic or alkaline your soil is. 7 is neutral. Below 7 is acidic. — 0 – 14
🌧️ Rainfall The average yearly rainfall in your region. Check your local weather data. mm 0 – 500
